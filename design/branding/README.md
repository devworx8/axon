# Axon Branding Concepts

This folder contains the first-pass Axon branding and UI direction assets:

- `axon-logo-concept-1.svg` — recommended primary logo (`Core Prism`)
- `axon-logo-concept-2.svg` — split A monogram
- `axon-logo-concept-3.svg` — orbital core / hero mark
- `axon_logo_and_ui_concepts.html` — concept board showing logo directions plus desktop/mobile UI composition ideas
- `axon_ui_concept_react.tsx` — React/Tailwind concept component
- `axon_ui_concept_tailwind.html` — standalone Tailwind HTML preview

Recommended next step: choose concept 1 as the base mark, then derive app icon, monochrome variants, and production dashboard/composer components from the concept board.
