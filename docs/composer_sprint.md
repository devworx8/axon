# Axon Composer Sprint

This sprint upgrades the Console composer into a true operator dock.

## Goals

- Introduce a structured composer with grouped actions
- Surface document/invoice creation actions visibly
- Expose Local Tools clearly
- Prepare for Code Action / self-coding flows
- Improve mobile and desktop parity

## Interaction Model

### Modes
- Ask
- Agent
- Code
- Invoice
- Research
- Local Tools

### Plus Menu Groups
- Create: Invoice, Quote, Mission, Prompt, Note
- Local Tools: Browse Files, Read File, Write File, Desktop Preview, Terminal, Scan Workspace
- Code Actions: Plan Patch, Preview Diff, Apply Change, Verify, Rollback
- Research: Deep Research, Use Template, Save to Memory, Search Resources

## UI Layers

1. Status strip
2. Action row
3. Main input
4. Utility row

## Expected Result

The bottom console area should feel like Axon's command dock, not just a text box.
